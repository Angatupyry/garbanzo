Ejercicio 1: "seleccionar" la caja vía id

En la consola, trata de seleccionar la cajita utilizando document.getElementById y guardala en una variable

🔎 Pueden buscar el id de la cajita desde DevTools.

✅ Para corregir: escriban pintarCajitas(variable) en la consola. Se debería pintar de azul.

Más información: MDN

Ejercicio 2: "seleccionar" las cajas vía clase y pintar la segunda

🧷 La función document.getElementsByClassName nos permite seleccionar todos los elementos que tengan cierta clase CSS, por ejemplo: document.getElementsByClassName('ej-2')

En la consola, trata de seleccionar las cajitas utilizando document.getElementsByClassName con la clase "ej-2". Guarden el resultado en una variable.

❓ ¿Qué pasa si escribimos variable[0] o variable[1] en la consola? (cambien variable por el nombre que eligieron para guardar los elementos!)

❗ La variable guarda todos los elementos encontrados con la misma clase en una especie de array. En este caso hay dos cajitas.

✅ Para pintar la segunda: escriban pintarCajitas(variable[1]) en la consola. Se debería pintar sólamente la segunda.

Más información: MDN

Ejercicio 3: "seleccionar" cajitas como si fuera CSS

🧷 La función document.querySelector nos permite utilizar selectores CSS para seleccionar elementos, por ejemplo: document.querySelector('.text-box')

🧷 Esta función nos devuelve el primer elemento que hace match con el selector. Podemos también utilizar document.querySelectorAll para traer una lista con todos los matches.

🧷 En CSS, existía un pseudo-selector llamado :nth-child(), el cual nos permitía seleccionar un enésimo elemento

En la consola, selecciona las cajitas utilizando document.querySelector('.ej-3:nth-child(n)') donde n es el número de cajita que quieren seleccionar: prueben varias!

❗ Podemos utilizar todo lo aprendido de CSS para seleccionar elementos: si lo podemos pintar lo podemos tomar desde JavaScript.

Más información: MDN

❔❔❔❔ ¿Siempre hay que poner document para seleccionar elementos?

👉 ¡No! podemos ejecutar estas funciones a partir de cualquier elemento del documento

👉 Por ejemplo, si creamos una variable llamada contenedor y guardamos en ella un div con varios elementos internos, podríamos hacer contenedor.querySelector() y reducir nuestra búsqueda a elementos internos a contenedor

Ejercicio 4: crear nuestra propia cajita!

🧷 La función document.createElement nos permite crear elementos, solo tenemos que decirle cuál, por ejemplo: document.createElement('div') crearía un div.

🧷 Esta función devuelve el elemento creado, pero no lo coloca en la página, por eso puede ser importante guardarlo en una variable para mantener una referencia.

🧷 Una vez creada la cajita, podemos agregarle clases utilizando classList.add()

🧷 Por otro lado, la función document.appendChild nos permite adjuntar un elemento al documento, o a cualquier otro elemento HTML.


Instrucciones:

👉 Crea una variable llamada contenedor cuyo valor sea igual a document.querySelector('.contenedor-cajitas'). Eso te debería ayudar a seleccionar el contenedor de la cajita verde que está abajo.

👉 Crea una variable llamada cajita, cuyo valor sea un nuevo div

👉 Agrégale la clase test-box a la variable cajita utilizando cajita.classList.add('test-box')

👉 Adjunta cajita al contenedor utilizando contenedor.appendChild(cajita)

❓ ¿Qué pasa si volvemos a ejecutar contenedor.appendChild(cajita)?.

❓ ¿Qué pasa si tratamos de adjuntar cajita a otro elemento?

❗ Cajita es un único div, no puede estar en más de un lugar a la vez

Más información: createElement(), classList, appendChild()

Ejercicio 5: modificar los estilos de la cajita, con JavaScript!

🧷 Para cualquier elemento interno del HTML, podemos acceder al objeto style.

🧷 Pueden seleccionar la cajita de abajo con document.getElementById('ej-5') y probar en consola agregarle .style para ver el listado de propiedades.

🧷 Si podemos acceder al objeto de propiedades, entonces podemos cambiarlo! Por ejemplo cajita.style.background = 'red' cambiaría el fondo de verde a rojo.

🧷 Por otro lado, la función document.appendChild nos permite adjuntar un elemento al documento, o a cualquier otro elemento HTML.


Instrucciones:

👉 Selecciona la cajita y guardala en una variable con cualquier nombre.

👉 Cambia algún valor CSS de la cajita, por ejemplo el borde o el color de fondo

🧷 Nota: las propiedades con guíon (ej: border-radius) cambian de nombre a su versión camel case, por lo que "border-radius" pasaría a ser "borderRadius" y "background-color" sería "backgroundColor"

🧷 Nota: es importante recordar las comillas en el valor (ej: '20px')

Más información: MDN

Ejercicio 6: destruir la cajita malvada!

🧷 Podemos ejecutar la función .remove() en un elemento para aniquilarlo.

🧷 Esta función no elimina el elemento de la faz de la tierra, simplemente lo saca del HTML, pero sigue quedando en la memoria de la máquina. Esto significa que podríamos volver a agregarlo con appendChild().


Instrucciones:

👉 Selecciona la cajita roja y guárdala en una variable.

👉 Llama a la función remove en la cajita, por ejemplo: cajaRoja.remove()

👉 Resulta que la cajita no era tan malvada 👀, vuelve a agregarla al contenedor de donde salió (van a tener que seleccionar el contenedor)

👉 Utiliza la función pintarCajitas(cajaRoja) para sacarle la maldad

Más información: MDN

Ejercicio 7: calma a la cajita!

🧷 Podemos ejecutar código cuando ocurre algún evento en particular, por ejemplo cuando hacemos click.

🧷 La forma de escribir un listener es la siguiente: elemento.addEventListener('nombre-evento', function() { código a ejecutar }) donde elemento es cualquier elemento interno del HTML.


Instrucciones:

👉 Selecciona la cajita movediza y guárdala en una variable.

👉 Agregale un event listener para el evento 'click' y dentro de la función, agregale la clase 'calm-box' a la caja

Más información: MDN

Ejercicio 8: que crezca la cajita!

🧷 La función que le pasamos al event listener puede ser creada en la misma declaración (como en el ejercicio anterior) o puede ser una ya existente.

🧷 En este ejercicio, existe una función llamada agrandarCajita() a la cual le podemos pasar una cajita para hacerla crecer de a 10 píxeles cada vez que se ejecute.


Instrucciones:

👉 Selecciona la cajita verde y guárdala en una variable.

👉 Agregale un event listener para el evento 'click' y en lugar de crear una función nueva, pasa como segundo parámetro agrandarCajita (sin paréntesis ni variable)

👉 Si le dan click a la cajita, se va a agrandar! Traten de que sea tan grande como el borde punteado.

Ejercicio 9: rotar la cajita!

🧷 Esta vez, vamos a tener que crear una función que rote la cajita cambiando su propiedad de transform.


Instrucciones:

👉 Selecciona la cajita verde y guárdala en una variable.

👉 Agregale un event listener para el evento 'click' y en la función, cambien el valor de style.transform de la cajita por 'rotate(120deg)'

Instrucciones:

Ejercicio 11: Suma en la cajita

🧷 Nota: usar la función parseFloat(input.value) para transformar el valor de los inputs en números.


Instrucciones:
